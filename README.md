## ⚡ Core Features

- **Advanced RAG System**: Strictly processes official university PDFs and documents.
- **Institutional Identity**: Hardened against prompt injection to ensure TUK-only responses.
- **Contextual Memory**: Remembers user preferences and academic interests.
- **Modern Interface**: A premium, responsive chat experience.

---

## 🛠️ Stack

- **Framework**: Next.js 15+ (React 19)
- **Database**: MongoDB Atlas
- **Inference**: Groq (Llama 3)
- **Knowledge Base**: PDF / JSON / TXT Document Ingestion

---

## 🚀 Quick Start

1. **Install**: `npm install`
2. **Environment**: Setup `.env.local` with your API keys.
3. **Data Ingest**: Visit `/api/admin/ingest` to sync your documents.
4. **Run**: `npm run dev`
